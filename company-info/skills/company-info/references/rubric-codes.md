# Belgian Annual Accounts — Rubric Codes

Common rubric codes from the MAR (Minimum Algemeen Rekeningenstelsel) used in
annual accounts filed with the NBB. Codes follow the Belgian accounting schema.

## Key Ratios

Some commonly derived figures:

| Metric | Calculation |
|--------|-------------|
| EBITDA | `9901` + `630` (operating profit + depreciation) |
| Net margin | `9903` / `70` × 100 |
| Current ratio | `29/58` / `42/48` |
| Solvency | `10/15` / `10/49` × 100 |
| Debt ratio | `17/49` / `10/49` × 100 |
| ROE | `9903` / `10/15` × 100 |
